## 搬瓦工 KVM OVZ 到底是什么？一句话说清楚

**KVM 和 OVZ（OpenVZ）是两种不同的虚拟化技术**，决定了 VPS 底层运行的方式。

OVZ 是操作系统级虚拟化，多个 VPS 实例共用宿主机的内核；KVM 是完全虚拟化，每个实例各自独立，互不干扰。换个说法：OVZ 像合租公寓共用墙体，KVM 像独立小户型。

对普通用户来说，KVM 的实际意义是：

- 支持 BBR 加速（OVZ 不行，或者要绕很远的弯子）
- 可以自己装内核、改系统参数，灵活度高
- 资源独立，邻居疯狂跑任务不会明显拖累你

OVZ 当时能卖是因为早几年有价格优势。但现在的情况是，OVZ 已经不存在了。

## KVM 套餐里各机房怎么选？

买了 KVM 常规套餐之后，可以在 KiwiVM 后台自由切换机房，但有个要注意的点：

**迁到 CN2 机房（DC3），流量变成三分之一。**

比如你买的是 20G KVM，每月有 1TB 流量，迁到 DC3 CN2 机房之后就只剩 333GB。如果你流量用得多，这个损耗不小。

推荐机房：DC2 QNET（洛杉矶）、DC4 MCOM（洛杉矶）、DC8 ZNET（洛杉矶）、弗里蒙特。这几个流量不打折，对大多数人够用。
